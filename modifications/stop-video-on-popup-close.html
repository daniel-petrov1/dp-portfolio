Stopping video when closing a popup in Webflow

Step by step

Step 01

Create your modal

Set the modal window sections/divs to position: fixed, add a video, and configure the visual styling and opening/closing animations.

Your modal must include at least the following elements:
1. the main container/div/section of the modal,
2. a close button (or another element to close the video),
3. and the video.

Step 02

Assign the class auto-stopper to the main container / div / section of the modal.

Step 03

Assign the class auto-stop to the closing background.

If your modal closes only when clicking the cross or a dedicated button â€“ skip this step.

Step 04

Assign the class auto-stop to the closing cross (close button).

Step 05

Assign the class auto-stop to the video itself.

Code

<script>
  // Modification for Webflow by Daniel Petrov
  
  var autoStopperBtn = document.querySelectorAll('.auto-stop');

  for (var i = 0; i < autoStopperBtn.length; i++) {
    (function(i) {
      autoStopperBtn[i].onclick = function () {
        var autoStopperModule = autoStopperBtn[i].closest('.auto-stopper');
        var autoStopperFrame = autoStopperModule.querySelector('iframe');
        var autoStopperSrc = autoStopperFrame.src;
        autoStopperFrame.src = '';
        autoStopperFrame.src = autoStopperSrc;
      }
    }(i));
  }
</script>
